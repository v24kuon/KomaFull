# KomaFull MVP プラン遵守と「停止」の徹底

このプロジェクト（KomaFull MVP）では、以下の運用を絶対的なルールとする。

1. **プランの絶対性**: `@/.cursor/plans/komafull_mvp_plan_25de4b13.plan.md` に記載された To-dos が全ての作業のマスターである。
2. **「止まる」指示の厳守**: To-do に「※ここで必ず止まって社長の確認を待つ」とある場合、そのステップを完了（commit & push 等）した時点で**即座に応答を返し、次の作業を一切行わない**。
3. **独断の禁止**: プランにないマイグレーションの実行、ファイルの削除、ライブラリの追加などを勝手に行わない。
4. **確認の徹底**: 疑問がある場合は、作業を継続する前に必ずユーザー（社長）に確認する。

このメモリは、エージェントがセッションを跨いでプランの進行状況と「停止」のルールを忘れないようにするためのものである。
