# リスク・品質知見 (Risk Knowledge)

このファイルは **監査参謀C (bucho-subordinate-risk)** が管理し、セキュリティや品質管理上の重要事項を記録します。

## セキュリティ鉄則
- 会員と管理者が同一テーブルのため、認可（Policy/Gate）のテストを最優先事項とする。
- Webhook受信ログ（`webhook_logs`）による監査証跡の完全保持。
- 管理者による重要操作（返金・強制昇格等）の二重確認または監査ログ。

## 品質・テスト方針
- 決済の冪等性テスト（重複リクエスト）を必須とする。
- Stripe API ダウン時のフォールバック挙動の検証。
